# DeepChappie
该项目的立意在于自己来实现大量的linux底层接口，并通过这些接口封装变成一个可执行文件，这个可执行文件。
其本意就是想将所有的linux的接口都完整地调用起来。

主要业务有：
1.多媒体业务
2.存储业务
3.网络业务



# 系统架构图

整体架构如下：

HAL->Framework->App

HAL层主要负责最基础的Linux操作的封装与Linux部分的底层操作

Framework层主要负责降底层功能组合起来，形成一个相关的功能模块或者单元

App负责相关的业务逻辑编写，当然App可以直接调用HAL
